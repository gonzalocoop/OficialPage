Proyectos Universitarios

Juego del Molino (Ciclo 1)

C++

Juego del Molino implementado en C++, donde dos jugadores colocan y mueven fichas en un tablero de 24 posiciones. El objetivo es formar "molinos" (líneas de tres) para eliminar las fichas del oponente.

Clasificación de Relaciones (Ciclo 3)

C++

Proyecto en C++ para clasificar relaciones de subconjuntos, determinando si son reflexivas, irreflexivas, simétricas, antisimétricas o transitivas, basado en una matriz de relaciones de entrada.

Gráficas de Componentes Conexas (Ciclo 4)

C++

Proyecto en C++ para dibujar grafos a partir de datos ingresados manualmente o generados aleatoriamente, representándolos visualmente y determinando sus componentes conexas mediante algoritmos de búsqueda.

Página de MedicVerse Solo Visual (Ciclo 4)

HTML | CSS | JS

Visita

Prototipo de una página web en HTML, CSS y JavaScript para un consultorio médico, donde los usuarios pueden visualizar un formulario de citas con médicos de medicina general, pero sin funcionalidad backend.

Página de Barbería con Windows Forms (Ciclo 5)

C# | SQL

Aplicación de escritorio para una barbería desarrollada en C# con Windows Forms y SQL Server, que permite gestionar clientes, citas y servicios de corte de cabello de forma funcional, almacenando los datos en una base de datos.

Página Funcional para Clases de Relajación (Ciclo 6)

HTML | CSS | TS | Java | PostgreSQL

Visita

Plataforma web funcional para clases de relajación y mindfulness, desarrollada con Java (Spring Boot) en el backend, Angular en el frontend y PostgreSQL como base de datos, permitiendo gestión de usuarios, reservas de clases y seguimiento del progreso (BackEnd Apagado).
